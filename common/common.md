# Agent Rules

## Writing

+ Prefer fluent English.
+ Use less Emojis, better not use at all.

+ Fixes grammar and spelling mistakes for documents and comments if read.

## Commentary

+ Comment should have proper grammar and punctuation.

## Code Style

+ 80 characters per line limit.

## Software Design Principles

+ Fail fast. Instead of explicit checking, just assume errors won't happen.
+ Use log with proper log level instead of print.
+ Only update minimal necessary code when fixing things.
