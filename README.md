# Agent Rule Repository

Team-managed repository for code agent rules.
Contains various rules for code agents in different scenarios.

## Usage

Run code agent from this project and ask it to update a `AGENTS.md` file in
another project.

(See [agent file](./AGENTS.md) for more details.)

## Table Of Content (partial, can be incomplete)

- `common/`: General rules that apply to various scenarios.
- `version-control/`: Actions with version control systems like git.

- `<programming-language>/`: Language-specific rules.
