# Agent Rule Repository

Team-managed repository for coding agent rules.
Contains different kinds of rules for coding agents in different scenarios.
When actually using it, only add the revalant parts to the project.

## Content (partial, can be incomplete)

- `common/`: General rules that apply to various coding agents. 
- `<programming-language>/`: Language-specific rules.

## Usage

Run AI agent from the target project then say about this repository.
