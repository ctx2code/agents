# Agent Rules

## Use This Project

- This project means to be a config house of `AGENTS.md` file.
- Read the content of these markdown files then choose suitable ones.
- Update the target project's `AGENTS.md` file with the content you choose.

### On Update

- Check the last modification time of dest `AGENTS.md` file.
- Check recent commits since then on both sides.
- See what new things you want to add.
